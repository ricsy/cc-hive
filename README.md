# cc-hive

Curated collection of practical plugins

## Usage

* add marketplace

```bash
claude plugin marketplace add ricsy/cc-hive
# or claude code terminal
/plugin marketplace add ricsy/cc-hive
```

* update marketplace

```bash
claude plugin marketplace update ricsy/cc-hive
```

* delete marketplace

```bash
claude plugin marketplace remove ricsy/cc-hive
```
